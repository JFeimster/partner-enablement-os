# Darwin Hanneman Launch Kit Index

Darwin Hanneman's launch kit contains partner-specific variants and launch assets generated from reusable Partner Enablement OS patterns.

## Source-to-Variant Rule

Canonical source assets live in:

```text
referral-partner-assets/
scripts/
trackers/
templates/
```

Darwin-specific variants live in:

```text
launch-kits/darwin-hanneman/
```

Use canonical assets for future partners first, then generate partner-specific variants only when partner personalization is needed.
