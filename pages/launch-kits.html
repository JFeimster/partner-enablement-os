Clone the System

How launch kits should be cloned for future Partners.

Start with Darwin’s structure, then swap the Partner profile, referral lanes, recommended assets, and launch objectives. Keep the compliance language intact unless it has been reviewed.

  • Define the Partner lane and audience.
  • Map the best-fit referral sources.
  • Attach only relevant scripts, trackers, and handouts.
  • Use safe language around possible funding paths.
  • Include the default funding disclaimer anywhere funding options are discussed.

Do Not Clone This

  • Guaranteed approval language
  • Specific rate promises
  • Funding amount promises
  • Timeline guarantees
  • Claims that everyone qualifies
