Asset Action Rules

Source, export, review, then use.

GitHub markdown is the canonical source. Google Drive is the exported and shareable layer. If an asset is marked needs cleanup, review it before field use.

  • Use GitHub source files for canonical edits.
  • Use raw markdown links for copying source into Notion, Docs, or internal notes.
  • Use Google Drive links for PDFs, Docs, videos, and shareable exports.
  • Check permission and compliance status before sharing Drive assets publicly.
