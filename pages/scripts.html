Copy/Paste Workflow

Copy the script. Do not copy bad assumptions.

Scripts are starting points. Adapt them to the person, lane, referral source, and actual business context. Keep the compliance-safe language intact.

  1. Open the source script. Use GitHub as the canonical file.
  2. Copy raw markdown if needed. Use the raw markdown button for fast transfer.
  3. Adjust the audience. Make the script fit the business owner or referral source.
  4. Remove unsafe claims. Do not promise approval, funding, rates, terms, amounts, timing, qualification, revenue, income, or outcomes.
  5. Save the adapted version. Put it in Notion, Docs, CRM notes, or the appropriate repo path.

Safe Usage Reminder

Use scripts to create clear conversations. Do not use scripts to imply guaranteed approvals, guaranteed funding, specific terms, specific timing, or universal qualification.
