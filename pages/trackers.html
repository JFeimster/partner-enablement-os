Daily Tracker

How to use the Daily Outreach Tracker.

Use this daily to log who you contacted, what was discussed, what asset was shared, and the next action. Keep it simple enough to actually use.

  • Log the person or organization contacted.
  • Record the referral lane or Partner type.
  • Note what was discussed.
  • Track the asset shared.
  • Write the next action and follow-up date.

Daily Rule

Track actions, not fantasies. Five useful actions beat twenty vague intentions wearing a productivity costume.

Weekly Tracker

How to use the Weekly Activity Tracker.

Use this weekly to review outreach volume, referral conversations, stuck follow-ups, and next-week priorities.

  • Review completed outreach and follow-up.
  • Identify stuck conversations.
  • Check which referral lanes are producing useful conversations.
  • Decide what assets need improvement.
  • Set next-week priorities.

Weekly Rule

A tracker nobody reviews is just a spreadsheet wearing tactical pants.

Tracker Workflow

Use the tracker without overcomplicating the machine.

Start with the actual tracker source. Copy it into the working tool your Partner already uses. Review the work weekly.

  1. Open the tracker. Use the GitHub source or raw markdown link.
  2. Copy into Notion, Google Docs, Sheets, or CRM notes. Use the format that will actually get updated.
  3. Fill it in daily or weekly. Track real actions, conversations, and next steps.
  4. Review next actions. Identify follow-up, stuck conversations, and missing context.
  5. Update Partner follow-up. Turn the review into next actions, not decorative data.

Compliance note: Trackers measure activity and follow-up discipline. They do not guarantee approvals, funding, revenue, income, applications, or outcomes.
