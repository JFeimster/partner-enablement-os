# Partner Enablement Prompt Chain

Use this chain to create partner launch assets in a controlled sequence.

## Master Instruction

You are creating a partner enablement asset for Moonshine Capital.

Use the available meeting notes, partner profile, source docs, and prior assets. Keep the output practical, compliant, relationship-first, and ready to use.

Avoid:

- guaranteed approvals
- guaranteed funding amounts
- guaranteed income
- misleading terms
- generic motivational filler

## Default Sequence

1. Create partner profile.
2. Create 30-day launch plan.
3. Create weekly activity tracker.
4. Create referral one-pagers.
5. Create outreach scripts.
6. Create printable handouts.
7. Create Flash-UI landing page prompts.
8. Create Wix booking/event copy.
9. Create HubSpot follow-up tasks.
10. Create NotebookLM prompt pack.

## Per-Deliverable Output Footer

Each generated asset should end with:

- Suggested GitHub path
- Suggested Notion status update
- Suggested export filename
- Next recommended deliverable

## Compliance Reminder

Use language like:

- explore funding options
- may be available
- based on business profile
- provider criteria apply
- approval, terms, amounts, and timing are not guaranteed

Avoid language like:

- guaranteed approval
- everyone qualifies
- guaranteed same-day funding
- any amount available
- credit does not matter
