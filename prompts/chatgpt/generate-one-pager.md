# Generate One-Pager Prompt

Use this prompt to generate a partner, referral partner, audience, or niche one-pager.

## Asset Status

- Status: Ready
- Owner: Moonshine Capital
- Audience: Internal Operator
- Platform: ChatGPT
- Last reviewed: 2026-05-03

## Objective

Create a concise one-pager that helps a partner explain who they help, what problem they address, and what the next responsible step is.

## Inputs Needed

- Partner name
- Target audience
- Audience pain points
- Partner positioning
- CTA or application link
- Any source notes

## Prompt

```text
Create a markdown one-pager for Moonshine Capital.

Partner: [Partner Name]
Target audience: [Audience]
Audience pain points: [Pain Points]
Partner positioning: [Positioning]
CTA/link: [CTA or Link]
Source notes: [Notes]

The one-pager should include:
1. Clear title
2. Quick summary
3. Who this helps
4. Common situation or pain point
5. How Moonshine Capital helps
6. Good-fit signals
7. Simple referral language
8. Next step
9. Responsible-use/compliance note

Make it clear, practical, and easy to turn into a PDF, Google Doc, Notion page, Wix section, or Canva flyer.

Do not guarantee approval, terms, amounts, rates, income, or timelines. Use responsible language and note that provider criteria apply.

End with:
- suggested GitHub path
- suggested Notion status update
- suggested export filename
- next recommended deliverable
```

## Suggested GitHub Output Path

```text
launch-kits/{partner-name}/one-pagers/{audience}-one-pager.md
```
