# Trucking Referral Assets

Canonical source assets for owner-operators, trucking companies, dispatch networks, repair shops, commercial truck contacts, fleet operators, and transportation-related referral channels.

## Audience

Referral sources connected to trucking businesses that need capital for repairs, fuel, insurance, payroll, equipment, trailers, freight timing, or short-term cash-flow gaps.

## Core Pain Points

- A parked truck does not pay bills.
- Repairs, fuel, insurance, payroll, and load timing can create urgent capital pressure.
- Freight payments and invoices may lag behind operating costs.
- A funding review can help truckers keep revenue moving when cash timing is ugly.

## Canonical Assets

Recommended source files:

- `one-pager.md`
- `printable-handout.md`
- `outreach-script.md`
- `follow-up-script.md`
- `landing-page-copy.md`

## Partner Variant Strategy

Use these canonical assets as the source. Create a partner-specific version under `launch-kits/{partner-name}/` only when adding partner contact info, QR/application link, local market notes, partner-specific lanes, or launch-specific positioning.
