## Design Direction

Use a clean fintech-style layout with strong contrast, simple cards, clear CTAs, and no clutter.

Suggested style:

- Professional
- Trust-building
- Mobile-friendly
- Direct
- Easy to skim

## Required Compliance Language

Include a clear note that approval, terms, amounts, and timing are not guaranteed and provider criteria apply.

## Prompt

```text
Create a high-converting landing page for [Partner Name], a relationship-first commercial capital partner backed by Moonshine Capital.

Audience: [Audience]
Goal: [Goal]
Primary CTA: [CTA]
Tone: Professional, direct, trustworthy, and relationship-first.

The page should include:
- Hero section
- Trust section
- Common problem section
- Who this helps section
- How it works section
- Referral partner section
- FAQ section
- Final CTA
- Compliance footer

Use responsible funding language. Do not guarantee approval, amounts, terms, or timelines.
```

## Publishing Notes

- If publishing to Wix, adapt into sections and buttons.
- If publishing to Vercel, preserve page sections as components.
- If publishing with Flash-UI, keep the prompt clean and design-forward.
