# Trackers

Canonical reusable trackers live here.

Use this folder for tracker content that can support multiple partners before it becomes partner-specific.

## Rule

```text
trackers/ = canonical reusable tracker templates
launch-kits/{partner-name}/trackers/ = partner-specific tracker variants only when materially customized
```

Do not store full reusable trackers only inside `templates/`. Templates should provide structure. Usable tracker source files should live here.

## Active / Planned Files

- `weekly-activity-tracker.md`
- `daily-outreach-tracker.md`
- `referral-partner-tracker.md`
- `application-pipeline-tracker.md`

## Partner Variant Rule

Create partner tracker variants only when the tracker includes:

- partner-specific target audiences
- partner-specific weekly goals
- custom activity targets
- partner launch-plan references
- partner coaching notes
- partner-specific metrics

## Export Targets

Trackers can be adapted into:

- Google Sheets
- Notion databases
- printable PDFs
- weekly review agendas
- HubSpot task review notes
