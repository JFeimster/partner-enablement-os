# Daily Outreach Tracker

**Asset Type:** Canonical tracker  
**Audience:** Partners / operators

## Purpose

Use this tracker to measure daily outbound activity, follow-ups, referral conversations, and application movement.

This is designed for simple daily execution. No dashboard cosplay. Count the work, spot the bottlenecks, and fix what is not moving.
